# Desafio - Boocamp 4ª edição

Este é um projeto desenvolvido para o desafio do Boocamp 4ª edição da IBID. O projeto consiste em criar um CRUD para o cadastro de um produto, usando C# e Visual Studio 2022.

## Tecnologias

- C#
- Visual Studio 2022
- Console Application (.NET Core)

## Requisitos

- Ter o Visual Studio 2022 instalado na máquina
- Ter o .NET Core 3.1 ou superior instalado na máquina
## Instruções

Para executar o projeto, siga os seguintes passos:

1. Clone este repositório na sua máquina local, usando o comando `git clone https://github.com/seu-usuario/seu-projeto.git` ou usando o GitHub Desktop.
2. Abra o Visual Studio 2022 e selecione **Arquivo > Abrir > Projeto/Solução**.
3. Navegue até a pasta onde você clonou o repositório e selecione o arquivo **ConsoleApplication.sln**.
4. Aguarde o Visual Studio carregar o projeto e suas dependências.
5. Pressione a tecla **F5** ou clique no botão **Iniciar** para executar o programa.
6. Uma janela do console deve aparecer com as opções do CRUD para o cadastro de um produto.
7. Digite o número da opção que deseja realizar e pressione a tecla **Enter**.
8. Siga as instruções na tela para adicionar, remover, editar ou visualizar os produtos cadastrados.
9. Para sair do programa, digite **0** e pressione **Enter**.
